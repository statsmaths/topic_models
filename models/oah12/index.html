OAH Theme Visualizer

This website allows participants to explore related events across conference formats by theme. Using a computational method called topic modeling, which identifies latent themes in a textual corpus, each event is assigned to a topic. Each topic is a list of words that tend to co-occur in the same documents. These topics can be understood as themes or discourses. Select a theme and see the related events. Select the event and see corresponding information such as speaker, date, and time.

  • Topic Grid: Visualizes thirteen themes that are prevelant throughout the conference. The first seven associated words are provided. Click on the theme to see related events.
  • Topic List: Lists the same twelve topics avaiable in the grid. For each theme, the most prevelant words are listed. The bar graph and percentage on the far right give the proporation of the events that are assigned to the theme.
  • Word index: List of words that are prominent in the twelve topics.

Within each theme, there is a list of the top 20 events alongside the percentage with which that event corresponds with the theme. The top words lists the words that define the theme. Click on the event to see more details including the data, time, event type and abstract. Content, participants, and schedule are accurate as of 10 November 2017.

The OAH Theme Visualizer is built by Lauren Tilton and Taylor Arnold. Adapted from the Signs@40 interface developed by Andrew Goldstone, Susana Galán, C. Laura Lovin, Andrew Mazzaschi, and Lindsey Whitmore.
